# number-guessing-
Number Guessing Game is a simple interactive game where the player tries to guess a randomly generated number within a limited range. The app provides hints like “too high” or “too low” after each attempt. The player wins by guessing the correct number in minimum tries, making it fun, challenging, and great for improving logical thinking skills.
